Green VS Red

Rules:

1. Each red cell that is surrounded by exactly 6 green cells will also become green in the nextgeneration.

2. A red cell will stay red in the next generation if it has either 0,1,2,4,5,7 or 8 green.

3. Each green cell surrounded by 0,1,4,5,7 or 8 green neighbours will become red in the next generation

4. A green cell will stay green in the next generation if it has either 2,3 or 6 green neighbours.

Instructions

  • To generate a grid input rows and cols and press the button "Set Grid"
  • To generate a new grid first press the button "Clean Grid" to clear the old grid
  • To track a certain cell input coordinates x1 and y1 and press the button "Track cell"
  • To apply the rules to the grid press the button "Generate"
